# Github搜索

## 搜索框高级搜索限定词

- **in:**

  ```shell
  in:name example                      名字中有“example”
  in:readme example                    readme中有“example”
  in:description example               描述中有“example”
  ```

  

- **stars:**>1000      

  `star数目>1000`

- **forks:**>1000

  `fork数目>1000`

- **pushed:**>2019-09-01 

  `2019年9月1日后有更新。`

- **language:**java 

  `用Java编写的项目。`
